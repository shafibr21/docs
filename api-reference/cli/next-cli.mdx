# next CLI

The `next` CLI runs and builds Next.js applications. Common commands include:

- `next dev` — Start the development server.
- `next build` — Compile the application for production.
- `next start` — Start the production server for a built app.
- `next export` — Export a static HTML export of the app.

Usage examples

Start dev server:

```bash
pnpm next dev
```

Build for production:

```bash
pnpm next build
pnpm next start
```

Other commands and options

- `next telemetry` — Manage anonymous telemetry reporting.
- `next lint` — Run built-in linting (if configured).
- Various flags are available to customize build, output, and experimental features — consult your framework config and the CLI help output for full details.

Examples and configuration snippets are available in the official Next.js docs.
