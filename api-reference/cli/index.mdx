# CLI

Next.js ships with two main CLI tools:

- `create-next-app` — Quickly create a new Next.js application using the default template or an example repository.
- `next` — Run the Next.js development server, build your application, and perform other project tasks.

Use the pages linked below for reference and examples:

- [create-next-app](/api-reference/cli/create-next-app)
- [next CLI](/api-reference/cli/next-cli)
